먹고 싶은 거

먹고싶은거 청년다방 : 버갈튀김(꼭꼭), 차돌세트
닭발 : 의정부 만행닭발, 한신포차
버거 : 다운타우너(존맛), 길버트 버거, 의정부 파머스빌리지, 성신 버거파크, 모밀, 돈까스 : 안암 백소정, 혜화 정돈
카페 : 방배동 스케일즈 커피, 혜화 학림다방, 이태원 오지힐, 창동 꿀소담, 망원동 키오스크(프렌치토스트 꼭먹기), 홍대 낙랑파라, 쌍문 고르, 쌍문 오픈더도어, 익선 온화 양식 : 경복궁 힐사이드 테이블(주변 걷기 좋음), 건대 마초쉐프(마시쏘), 성신 문화식당, 익선 지오쿠치나, 혜화 파스타마켓, 메이빌(체인), 익선 백스트리트 피자, 연남 클래식당, 연남 포레스트, 홍대 요멘야고에몬, 홍대 피자 네버슬립드, 라라코스트(체인), 성신 리얼파스타, 연남 봄띠, 성수 다로베
샤브샤브 : 혜화 어바웃 샤브
일식 : 혜화 멘쯔(스키야끼)
베트남 : 합정 미에우(분짜 짱맛)
타이 : 어린이대공원역 타이인플레이트
빙수 : 혜화 타셴 복숭아빙수
오리 : 오리도니(체인)
아이스크림 : 노브랜드 초코 아이스크림
한식 : 서울대입구 킷사서울, 일미리 금계찜닭(체인), 광장시장 부촌육회, 인사동 도마
술집 : 홍대 머라이언(강추)
밥겸술 : 성신여대 동선물망초(하이볼 짱맛)
치킨 : 홍대 닭꼬얌, 홍대 더피자보이즈(피자집이지만 치킨이 맛있음)
떡볶이 : 혜화 불난집
타코 : 용산 토마틸로
